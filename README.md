# Despliegue-y-Aseguramiento-de-Servidor-Web-Empresarial-en-Linux-Ubuntu-Server-
Este laboratorio práctico simula un entorno de producción real mediante el despliegue desde cero de un servidor empresarial basado en Ubuntu Server 24.04 LTS virtualizado en VMware Workstation, operando el servidor de forma remota mediante SSH tal como se gestiona en entornos corporativos reales.
